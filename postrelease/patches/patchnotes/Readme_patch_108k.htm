program-fixes:


----------------------1.08k:------------------------

- This patch fixes occasional crashes during gamestart. Especially if you play under Windows 98 and use some weird graphiccard/driver combination there is a possibilty that the game crashes with an illegal call to a directx utility function named "D3DXGetDeviceCount". This misbehaviour is now corrected as we do now link the gothic.exe with the directx 8 libraries, which do better support the newer driver and graphiccards. We suggest to use this exe only if you encounter crashes during gamestart (in detail: if you encounter crashes with the radeon 8500 or the Geforce 3 under Win98) This version is not supported by Piranha-Bytes.
